# AG Manager 🚀

[Tiếng Việt bên dưới]

**AG Manager** is a professional, high-performance, and secure VS Code extension specifically designed for **Antigravity IDE** users. It provides a real-time dashboard to monitor your AI model quotas and manage automation features without compromising your security.

## 🛡️ Security First
Many community extensions for Antigravity have been found to contain malicious code. AG Manager was built with security as the top priority:
- **Zero Data Collection**: No data ever leaves your machine.
- **Local Discovery**: Uses secure, local-only discovery to link with your Antigravity instance.
- **Open Transparency**: Clean, readable code that respects your privacy.

## ✨ Features
- 📊 **Dynamic Dashboard**: Real-time progress bars for all your active models (Gemini, Claude, GPT-OSS).
- 🕒 **Accurate Resets**: Detailed "Refreshes in..." timers matching the Antigravity system accurately.
- ⚡ **Status Bar Integration**: Keep an eye on your most critical quota directly in the status bar.
- 🎯 **Auto Click**: Automatically click approval buttons (Run, Allow, Accept).
- 💎 **Premium UI**: Modern Glassmorphism design that feels native and premium.

# AG Manager 🚀 (Bản Tiếng Việt)

**AG Manager** là một tiện ích mở rộng chuyên nghiệp, hiệu suất cao và an toàn dành riêng cho người dùng **Antigravity IDE**. Nó cung cấp một bảng điều khiển thời gian thực để theo dõi hạn mức (quota) và quản lý các tính năng tự động hóa mà không làm ảnh hưởng đến bảo mật của bạn.

## 🛡️ Bảo mật là trên hết
Nhiều extension cộng đồng cho Antigravity đã bị phát hiện chứa mã độc. AG Manager được xây dựng với nguyên tắc bảo mật tối thượng:
- **Không thu thập dữ liệu**: Tuyệt đối không có dữ liệu nào rời khỏi máy bạn.
- **Khám phá nội bộ**: Sử dụng cơ chế tìm kiếm an toàn, chỉ chạy nội bộ để kết nối với Antigravity.
- **Minh bạch**: Mã nguồn sạch, tôn trọng quyền riêng tư của bạn.

## ✨ Tính năng nổi bật
- 📊 **Bảng điều khiển động**: Thanh tiến trình thời gian thực cho tất cả các mô hình đang hoạt động (Gemini, Claude, GPT-OSS).
- 🕒 **Thời gian Reset chính xác**: Đồng hồ đếm ngược "Refreshes in..." khớp hoàn toàn với hệ thống Antigravity.
- ⚡ **Tích hợp Thanh trạng thái**: Theo dõi hạn mức quan trọng nhất ngay tại thanh trạng thái.
- 🎯 **Auto Click**: Tự động nhấn các nút xác nhận (Run, Allow, Accept...).
- 💎 **Giao diện cao cấp**: Thiết kế Glassmorphism hiện đại mang lại cảm giác xịn xò và mượt mà.

## 📦 Cài đặt (Installation)
1. Trong Antigravity, nhấn `Ctrl + Shift + P`.
2. Gõ và chọn **"Install from VSIX..."**.
3. Chọn file `ag-manager-1.1.1.vsix`.
4. Nhấn biểu tượng Dashboard ở thanh bên trái để bắt đầu!
